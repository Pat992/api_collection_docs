This repository contains the generated Postman collections for my APIs published on [APIHUB](https://apihub.cloud/user/Pat) and [RapidAPI](https://rapidapi.com/user/Pat92). Each folder corresponds to a different API.

### How to use a collection
1. Subscribe to the API you want to use.
2. Open Postman and click **Import**.
3. Choose one of the following methods:
   - Paste the collection URL directly.
   - Open the URL in your browser, right-click the page, save it as a JSON file, then import that file into Postman.
4. After importing, open the collectionâ€™s **Variables** tab and replace either the `x-rapidapi-key` if using RapidAPi or `X-APIHUB-TOKEN` if using APIHUB with your own credentials.

Once your variables are set, you can start exploring the endpoints and testing requests directly from Postman.
